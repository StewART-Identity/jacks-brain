`dsnr-idpr-app01` is the Identity Provisioning (IDPR) application server used to manage NetIQ Identity Manager configurations. [[collection/entities/netiq-identity-manager-designer]] is operated here with Administrator privileges to manage driver lifecycles.

Referenced in [[collection/sources/2026-04-25-driver-migration-husk]] as the target host for the HSCAD, UNTAD, and UNTADSTU driver migration under CHG0038717.
