Two ways to find things. Search the wiki for what you already have, then reach further afield.

- **[[search/web|Web]]** — send a query to the web for candidate sources you might want to catalog. Bring promising results back via Selection.
- **[[search/wiki|Wiki]]** — full-text search across every page in the wiki. Tags, titles, body. Start here.
