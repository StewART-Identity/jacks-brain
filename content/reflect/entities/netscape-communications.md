Netscape Communications Corp. was a pioneering internet software company headquartered in Mountain View, CA. Netscape engineers — including [[reflect/entities/tim-howes|Tim Howes]], M. Wahl, A. Coulbeck, and S. Kille — were central contributors to the LDAPv3 standards effort, co-authoring the cluster of December 1997 RFCs that defined [[reflect/concepts/ldap|LDAP]] version 3:

- RFC 2251 — LDAPv3 core protocol
- RFC 2252 — Attribute syntax definitions
- [[reflect/sources/2026-05-02-rfc2253-txt|RFC 2253]] — UTF-8 string representation of [[reflect/concepts/distinguished-name|distinguished names]]
- RFC 2254 — [[reflect/sources/2026-05-02-rfc2254-txt|String representation of search filters]]
- [[reflect/sources/2026-05-02-rfc2255-txt|RFC 2255]] — LDAP URL format
- RFC 2256 — Summary of X.500(96) user schema for LDAPv3
- RFC 2696 — [[reflect/sources/2026-05-02-rfc2696-txt|LDAP Control Extension for Simple Paged Results]] (co-authored with [[reflect/entities/microsoft|Microsoft Corp.]], September 1999)
