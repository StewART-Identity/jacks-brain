## Key Observations

### Windows Server Patching (Items 1, 4, 5)
December 2025 cumulative updates are outstanding across the [[entities/active-directory|Active Directory]] and IAM Windows fleet. Item 1 (Server 2022, KB5071547) is the highest-risk single remediation, affecting 35 assets including all AD domain controllers, IDM application servers, and jump hosts. Items 4 and 5 cover Server 2019 and 2016 hosts respectively.

### Linux Package Debt (Items 2, 9, 11, 15–20, 22, 23)
The entire Linux IAM fleet (~43 hosts spanning `ams-idpr-*`, `duo-idpr-*`, `rad-idpr-*`, `rllx-idpr-*`, SSO servers) carries a large backlog of Red Hat package updates. Items 15–20 are all sssd suite libraries (libsss_idmap, libipa_hbac, libsss_certmap, libsss_nss_idmap, sssd, libsss_sudo) affecting identical asset sets with identical risk scores — these would be resolved together by a single `yum update sssd` run.

### No-Solution Vulnerabilities (Item 3)
Item 3 flags 44 vulnerabilities across the full Linux IAM fleet for which no public fix exists. Recommended mitigations: apply hardening guides, remove affected software, restrict access, or decommission. This is the second-highest risk item and requires architectural/policy decisions rather than a simple patch.

### JVM Ecosystem (Items 6, 7, 14)
[[entities/idtree|IDM]] (idm-ardc-idv*, idmd-ardc-idv*, idmdidv*, idmpidv*) and IMAN servers require Azul Zulu and Oracle Java upgrades. SSO servers (sso1.gabdcn.unt.edu, sso1.ardc.unt.edu) require Apache Tomcat upgrade to 9.1.109.

### [[entities/duo|Duo]] Infrastructure
Duo production servers (duo-idpr-app01–06) appear in items 2, 3, 9, 11, 15–20, 22, 23 — virtually every Linux-side remediation. The Duo fleet runs on Red Hat and shares the same package debt as the broader IAM Linux fleet.

### TLS and Certificate Issues (Items 21, 24)
24 hosts (primarily AD domain controllers) still support the deprecated 3DES cipher suite. 35 Windows hosts require the Certificate Padding Check registry setting for CVE-2013-3900.

### CVE-2022-0001 — Branch History Injection (Item 25)
Spectre-BHI mitigation is needed on 47 Windows hosts (the broadest asset coverage of any item). Remediation requires registry keys to enable Intel processor mitigations:
```
FeatureSettingsOverride = 0x00800000
FeatureSettingsOverrideMask = 0x00000003
```
